# MailListRepo

Pagina web statica che, dati i 3 file Excel del gestionale (anagrafica, ordini, vendite), produce in browser i riepiloghi vendite/ordini aperti e una mail list scaricabile in CSV.

## Come si usa

1. Apri `index.html` nel browser (doppio click oppure via GitHub Pages).
2. Carica i 3 file Excel nei rispettivi campi e premi **Elabora**.
3. Consulta le tabelle a video e premi **Scarica CSV** per ottenere `maillist.csv`.

Tutto avviene client-side: i file non vengono mai caricati su nessun server.
